中东互联网独角兽深度报告：6家企业万亿估值背后的ICT版图

作者：Tech Observer | 2025年11月

本报告深入分析中东及北非（MENA）地区6家最具代表性的互联网独角兽企业——Careem、Noon、Shahid、Souq/Amazon、Talabat、Yalla——覆盖财务表现、ICT投入、云服务架构、数据中心策略及最新战略动向。6家企业总估值超过150亿美元，业务覆盖出行、电商、流媒体、外卖、社交娱乐等多个领域。

一、Careem：Uber系超级应用的云优先战略

估值：31亿美元（Uber收购价） | 总部：迪拜 | 覆盖：15国120城

核心发现

  • 超级应用战略：2021年启动，整合出行、外卖、支付（Careem Pay）、即时配送
  • e&战略投资：2023年获阿联酋电信集团4亿美元投资，加速超级应用业务扩张
  • 多云架构：800+微服务，主要基于AWS + GCP
  • AI驱动：欺诈检测（AWS Neptune图数据库）、需求预测、个性化推荐
  • 云优先策略：不自建数据中心，充分利用AWS/GCP中东（巴林、阿联酋）基础设施
  • 2025年营收预计超28亿美元

技术架构

AWS (核心) + GCP (大数据分析)
└── 800+ 微服务
    ├── AI/ML (GCP BigQuery)
    ├── 支付 (Careem Pay)
    └── 物流追踪

二、Noon：10亿美元帝国的Azure云转型

估值：10亿美元+ | 总部：迪拜 | 覆盖：阿联酋、沙特、埃及

核心发现

  • 创始背景：Emaar创始人阿拉巴尔联合沙特主权财富基金（PIF），2016年获10亿美元初始投资
  • 业务生态：Noon.com + Noon Food + Noon Minutes（15分钟快送）+ Noon Pay
  • 云服务：Noon Payments核心业务已迁移至Microsoft Azure
  • 2025年GMV预计50-70亿美元，接近盈利
  • 双重上市计划：未来18-24个月内在沙特和阿联酋IPO
  • 自动驾驶配送：2025年11月在阿布扎比启动试点

财务亮点

指标 数据
年度GMV 50-70亿美元
增长率 30%-50%年均
目标市场 阿联酋、沙特、埃及
ICT投入占比 15%-25%运营支出

三、Shahid：MBC集团流媒体野心的云端支撑

母公司：MBC Group（沙特PIF控股）| 总部：迪拜 | 覆盖：22个阿拉伯国家

核心发现

  • 市场地位：中东及北非地区最大的阿拉伯语VOD平台
  • 内容优势：拥有MBC集团全部广播电视内容独家版权，海量阿拉伯语原生内容
  • 沙特定位：作为沙特"2030愿景"数字娱乐战略核心，获得PIF强力支持
  • 多云策略：主要使用AWS + CDN加速，支撑大规模视频流
  • 用户规模：付费订阅用户持续增长，沙特用户占比超40%

四、Souq → Amazon.ae：亚马逊的中东棋局

母公司：Amazon（NASDAQ: AMZN）| 收购价：5.75亿美元（2017年）

核心发现

  • 历史地位：Souq.com被誉为"中东亚马逊"，2005年创立，是中东最早的电商平台之一
  • 亚马逊收购：2017年以5.75亿美元收购，随后品牌重塑为Amazon.ae/.sa/.eg
  • 技术整合：全面接入AWS，利用亚马逊技术栈升级物流、支付、推荐系统
  • FBA（Fulfillment by Amazon）：引入亚马逊成熟物流体系，大幅提升配送体验
  • AWS中东基础设施：Amazon.ae深度依赖AWS中东（巴林）区域

技术优势

亚马逊技术赋能：
├── AWS (云基础设施)
├── DynamoDB (数据库)
├── SageMaker (AI推荐)
└──物流网络（FBA）

五、Talabat：海湾外卖大战的技术底座

母公司：Delivery Hero（法兰克福上市）| 总部：科威特 | 覆盖：6国

核心发现

  • ** Delivery Hero子公司**：2020年被欧洲外卖巨头Delivery Hero收购
  • 技术栈：主要基于AWS构建，微服务架构，支持高并发订单处理
  • 市场覆盖：科威特、沙特、阿联酋、卡塔尔、巴林、阿曼
  • 骑手网络：在海湾六国运营超过10万名骑手
  • 快速扩张：通过并购整合本地外卖平台，形成海湾外卖垄断地位

六、Yalla：盈利奇迹背后的技术驱动

NYSE: YALA | 总部：迪拜 | 2020年上市（中东首家美股上市的独角兽）

核心发现

  • 独特商业模式：语音社交（Yalla）+ 游戏社交（Yalla Ludo），专注阿拉伯市场
  • 惊人盈利：净利润率常年保持40%以上，2024年营收超3亿美元
  • 轻资产技术：主要基于AWS构建，无需重资产物流基础设施
  • 本地化深度：产品完全针对阿拉伯用户设计，支持Arabic NLP
  • 2024年净利润超1.2亿美元

七、MENA互联网企业ICT共性分析

云服务商分布

云服务商 主要客户 数据中心位置
AWS Careem, Talabat, Yalla, Amazon.ae 巴林（中东）
Microsoft Azure Noon (Payments), Shahid 阿联酋
Google Cloud Careem (BigQuery) 阿联酋

关键趋势

  1. 多云策略：头部企业普遍采用2-3家云服务商，避免单点依赖
  2. 云优先（Cloud-First）：除Yalla外，主要企业均选择不自建数据中心
  3. AI驱动：所有企业均在推荐、欺诈检测、需求预测等领域深度应用AI
  4. 出海与中国：Noon进入中国市场，Talabat/Yalla关注海湾本地化

八、我的MENA观察

以下内容基于实地考察与行业交流。

MENA互联网市场是一个"政策驱动大于市场驱动"的特殊存在。沙特PIF的强势布局，让Shahid和Noon获得了其他国家无法想象的资本支持。而Yalla的盈利奇迹则证明，本地化做到极致，即使不依赖融资也能活得很好。

以上照片为作者在MENA区域实地拍摄

反观中国出海企业，在MENA区域的成功案例多集中在游戏（如米哈游、腾讯）和新能源（如Temu、SHEIN），纯互联网平台仍面临本地化壁垒。AWS和Azure的垄断性优势，也让中国云厂商的机会集中在混合云场景。
