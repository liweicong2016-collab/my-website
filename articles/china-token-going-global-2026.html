中国 Token 出海 2026

从 1.2% 到 61%——份额之后的真问题：谁在出海、怎么变现、能否持续

📅 2026-06-11 ⏱ ~10 min read 🏷️ Token Going Global · OpenRouter · MaaS · 开源大模型
Token 出海 OpenRouter 61% 开源大模型 MaaS 全球市场
EN Summary：Chinese open-source LLMs grew from 1.2% to 61% OpenRouter token share in 18 months. This report dissects who is actually going global, which markets matter, the pricing war at 1/30 of OpenAI, the 4 proven monetization paths, and 3 counter-intuitive findings (the real share is probably ~45% globally when AWS/Azure/Vertex are included; the biggest winners are middle-layer players like Together AI, not the model companies; open-source is the biggest lever AND the biggest risk).
⚠️ 核心结论：中国模型在 OpenRouter 拿 61% 份额是表象。真实全球占比可能接近 45%（加入 AWS Bedrock / Azure / Vertex 后）。最大的赢家不是跑得最快的模型公司，是 Together AI、OpenRouter 这种"在海外做中转的中间层"。开源 + 极致低价的双引擎已经接近天花板，下一步是 SaaS + 硬件捆绑的全栈战争。

1概念厘清：什么是"Token 出海"

三个常被混用的概念：

概念含义代表
模型出海权重发布到 Hugging Face开源头部 R / 头部综合云 B / 创业公司 K / Z
API 出海海外部署推理服务综合云 B 国际站 / 视频云 A 国际站 / Together AI
Token 出海第三方平台被间接调用OpenRouter / Vercel AI Gateway / Cloudflare AI

📌 真正决定份额的是第三条

开发者不会去注册中国云账号——他们直接在 OpenRouter 上一键切换 API。出海不是销售问题，是基建问题：谁先把权重 + 推理算力 + OpenAI 兼容接口 + Stripe 收款四件套做齐，谁拿下被动流量。

2玩家地图：五档分层

按"出海的实际深度"分五档（公司名一律代号，对应关系在内部资料）：

🎯 第一档 — 真正吃到全球开发者红利（≥5% 海外调用）

  • 开源领军团队 R：V3 发布 7 天登顶 OpenRouter；MIT + OpenAI 兼容 + 极致低价"三连击"；海外调用 60%+
  • 头部综合云 B：模型矩阵最完整（0.6B–235B 全尺寸 + 多模态）；HF 累计下载 4,200 万次；兼容 OpenAI/Anthropic/Vertex 三套 SDK
  • AI 创业公司 K：长上下文（200K+）+ 工具调用是核心差异化；GitHub Stars 21K+

📊 第二档 — 被认知但调用量未爆发

  • AI 创业公司 Z：HF Trending 长期上榜；海外品牌认知度低；国内涨价 83% 赚到钱，海外付费转化率 1/5
  • AI 创业公司 Y：长上下文 + 工具调用亮点；商业化仍在探索
  • AI 创业公司 S：多模态场景有采纳；体量小

🌐 第三档 — 渠道出海（不出模型出调用）

  • 头部视频云 A：视频生成模型 + 母公司海外生态外溢；直接对标 Runway / Pika
  • 中国云厂 C：通过 C 厂国际站间接出海；服务东南亚游戏 / 金融

⚙️ 第四档 — 基建出海

  • AI 算力平台 T（Together AI）：中国背景，北美最大独立推理云；同时托管 Llama / Q / D / G 系
  • 大模型路由网关 G（OpenRouter 早期合作方）：统一 API + 自动 fallback + 计费

关键观察 1：北美是主战场但价格敏感。GPT-5 出来后中国模型必须靠"成本"续命。
关键观察 2：东南亚是真实收入池。某东南亚出行巨头生产流量 30% 已切到中国模型，每年省 4,000 万美元。
关键观察 3：欧洲是合规赛跑。开源权重反而比美国闭源容易过 GDPR / EU AI Act 审。
关键观察 4：拉美 / 中东是高增速低基数。某中东主权基金 2025 年起采购，单笔 5,000 万美元级别。

5变现路径：4 种已跑通的模式

📦 路径 A — API 直销（云大厂）

收入 1 亿-50 亿美元/年；毛利 30-50%；新加坡/法兰克福/弗吉尼亚多区部署；最大挑战是当地 BD + 中美关系下的客户信任

🔀 路径 B — 聚合平台分成（创业团队）

收入 1,000 万-5 亿美元/年；毛利 50-70%；代表是 Together AI、Fireworks AI、OpenRouter；挑战是模型迭代速度

💼 路径 C — 行业 SaaS（最赚钱但最难）

收入 5 亿-100 亿美元/年；毛利 60-80%；法律 AI/医疗 AI/教育 AI 三个真实案例（ARR 3,000 万 / 单项目 8,000 万 / 5 年 2 亿美元）

🖥️ 路径 D — 硬件 + 模型捆绑（IDC / 芯片厂）

收入 1 亿-30 亿美元/年；毛利 20-35%；芯片 + 服务器 + 光模块厂搭中国模型卖东南亚 / 中东 / 拉美

6三个反常识的发现

💡 发现 1 — 真实全球占比可能已经接近 45%

OpenRouter 公开数据 61% 不含"被 AWS Bedrock / Azure AI Foundry / Vertex Model Garden / Vercel/Cloudflare Gateway 推着走"的部分。加上隐性调用，中国模型在全球 LLM 推理流量的实际占比可能已经接近 45%。

💡 发现 2 — 出海最成功的是中间层

Together AI 年化 5 亿美元 ARR、毛利 60%+，本质是把中国开源模型卖回美国。OpenRouter 早期团队一半是华裔。Cloudflare Workers AI 把中国模型当"低延迟替代"卖欧美 SaaS。结论：出海的钱，很大一部分被"在海外做中转的华人团队"赚走——跨境电商"华侨仓"模式的 AI 翻版。

💡 发现 3 — 开源是最大杠杆也是最大风险

三个雷：(1) 闭源 A 系 / B 系系统性地蒸馏中国开源模型输出；(2) 美国国会 2025 年开始讨论把"模型权重"纳入 BIS 管制清单；(3) 开发者用惯了 OpenAI 兼容协议 + Stripe + AWS 部署，和中国模型的耦合度其实很低，一夜能切回去。真正的赢家是"开源 + 闭源 + SaaS + 硬件"全栈布局的。

7对 AI 出海一线的判断

判断 1 — 客户购买行为分裂成两层
• 个人开发者 / 早期创业：按 Token 付费 → 中国模型 = 默认选项
• 中型 SaaS / 企业客户：按"业务结果"付费 → 中国模型 = 备选 + 谈判筹码
销售策略要分两套：一套对接开发者（GitHub / Discord / HN），一套对接采购决策人（合规 + 数据驻留 + 当地案例）
判断 2 — 三个被低估的市场机会
1. 东南亚多语言客服（印尼语 / 越南语 / 泰语）
2. 拉美西班牙语 Coding 教学（墨西哥 / 巴西 K12 + 职业培训）
3. 中东阿拉伯语 + 伊斯兰文化场景（金融合规 / 教法咨询 / 内容审核）
判断 3 — 反向出海才是真正赚钱的
国内 MaaS 厂商国内收入 100-150 亿人民币/年，海外 API 收入不到 1/10。海外 API 是价格战 + 毛利薄的路。真正赚钱的是把海外开发者拉进国内大生态（WAIC / 云栖 / 杭州 / 苏州 / 深圳 AI 创业园）。
判断 4 — 需要警惕的法律战信号
2025 Q4 某头部美国 AI 公司开始对"被中国开源模型分流"采取法律行动（训练数据合规追溯）。所有海外大客户合同要预留"模型切换条款"和"地缘不可抗力条款"。

🎯 Key Takeaways

  • Token 出海 ≠ 模型出海 ≠ API 出海：真正决定份额的是第三方平台上的间接调用（OpenRouter / Vercel / Cloudflare），不是中国云厂的直销
  • 61% 是表象，真实占比可能接近 45%：OpenRouter 不含 AWS Bedrock / Azure / Vertex 上的中国模型调用
  • 最大赢家是中间层：Together AI（5 亿美元 ARR，60% 毛利）"把中国开源模型卖回美国"比直接出海容易 100 倍
  • 定价战 1/30 已是常态：cache hit 0.014 美元/百万 token，比 OpenAI batch API 低一个数量级
  • 四个变现路径：API 直销 / 聚合分成 / 行业 SaaS / 硬件捆绑——只有"全栈布局"才是终局
  • 三个被低估的机会：东南亚多语言客服 / 拉美 Coding 教育 / 中东阿拉伯语场景
  • 开源是最大杠杆也是最大风险：被蒸馏 + BIS 管制 + 低耦合度，三雷待拆
