Three new angles: (1) unit economics of single token — 70-93% gross margin for vertical winners; (2) MaaS sales playbook — copy the form, not the essence; (3) 5 mechanisms sustaining 1/10 pricing: long-context tax, output inflation, scheduling discount, budget internalization, compliance premium + 3 China-specific overlays.

3Is the MaaS Sales Paradigm Worth Learning?

3.1 The Core of China's MaaS Sales Paradigm

The MaaS sales paradigm upgrade driven by a Leading Comprehensive Cloud B in 2025-2026:

Traditional Cloud Sales (Selling Resources) → MaaS Sales (Selling Tokens) → Agent Sales (Selling Capabilities / Selling Scenarios / Selling Outcomes)

Core Actions:

  1. Established several new MaaS sales teams to collaborate with existing direct sales teams on deals.
  2. Core KPIs: Discover usage scenarios + drive enterprise Agent adoption + increase Token consumption.
  3. Supporting Products: Industry Agent templates (e-commerce customer service / financial risk control / government Q&A / industrial documentation).
  4. Pricing Model: Annual framework agreements + Token packages + outcome-based sharing (tied to GMV / labor hours saved).

A variant from a Leading Video Cloud A (more aggressive): Model A directly uses a whitelist + annual framework model; queuing up to 10 hours during peak Spring Festival periods → forcibly filtering for the "highest-bidding + most deeply integrated" customers; stable supply of the "full-powered version" → prioritizing GPU resources for annual framework clients.

45 Mechanisms That Allow Sustained Operations at 1/10 the Price

This is the hardest-hitting section of this article. China's API prices are chronically 10x lower than overseas, and the market hasn't collapsed. It's not due to subsidies, but 5 underlying mechanisms that simultaneously boost volume + 3 China-specific overlays.

Mechanism 1: Long-Context Tax

Phenomenon: From 2025 to 2026, mainstream model context windows jumped from 8K to 128K / 1M / 10M.

  • A single RAG request stuffs in 500K-1M characters of context.
  • Token count per API call surged from ~1K to ~500K (500x).
  • Customers perceive "prices haven't changed" (API unit price dropped 5-10x), but actual spend increased 50x.

Token unit price fell 90%, but single-call amount grew 75x → customer's total bill actually grows 7.5x. Model providers (earning more per call) + downstream applications (better user experience) → win-win.

Mechanism 2: Output Inflation

Phenomenon: Models tend to "safely verbose output." A point that could be made in 50 words, AI will write in 500.

  • Early GPT-4: Averaged 200 tokens output / request
  • 2026 Q1 Mainstream models: Averaged 800 tokens output / request
  • Output tokens up 4x (even with the same input)

Output token pricing is typically 3-5x that of input (especially for Coding / Video generation). Customers pay more for "output tokens" → vendor revenue grows; but actual GPU computation hasn't grown 4x (caching / speculative decoding / quantization) → marginal cost growth is far lower than revenue growth. → For vendors: Expanding gross margin

Mechanism 3: Scheduling Discount

Marginal cost per token continuously declines through 4 technical means:

TechnologyPer Token Cost Reduction
CacheCache hit → drops 90%
QuantizationFP16 → INT8 / INT4 → drops 50-75%
MoE ArchitectureActive parameters squeezed to 3-5% → cliff dive
Speculative DecodingInference time drops 30-50%

Data (from the public technical report of the Domestic Open-Source Leader Team R):

  • Per token inference cost 2024 Q4: $0.0014
  • Per token inference cost 2026 Q1: $0.0002 (down 86%)
  • API price 2024 Q4: $0.27/million
  • API price 2026 Q1: $0.014/million (down 95%)
  • Price fell faster than cost → gross margin actually expanded

Mechanism 4: Budget Internalization

AI calls migrated from "R&D project expense" to "regular operational expense."

  • 2024: CFO perspective = "This is the R&D department's new toy"
  • 2026: CFO perspective = "This is like utilities + SaaS subscriptions + cloud resources"

Implications for Vendors

  • Internal customer approval paths shorten (IT department decision-making power increases).
  • Renewal rates increase (60-80% industry benchmark).
  • ARPU increases (CFO proactively raises budget caps).

Data: Median enterprise AI budget was $50K/year in 2024 → $500K/year in 2026 (10x increase). Main driver: The triple jump from "pilot" → "scaling" → "fixed expenditure."

Mechanism 5: Compliance Premium

Tokens that are auditable / source-tracked / privately deployed can still command a 30-100% premium.

Standard API price wars are intense (gross margins 30-50%), but compliance / private / industry-customized tokens remain high-margin (60-80%). The core profit source of MaaS sales is the compliance premium, not standard APIs.

6Counterarguments / Risk Factors

6.1 Three Hidden Concerns About Sustaining 1/10 Price

Concern 1: How Long Can the Subsidy War Last?

Current China API prices = cost + razor-thin profit (not a true market clearing price). If the industry consolidates one day (leaving 2-3 oligopolies), prices could rebound 5-10x. Analogy: China's cloud market saw a price war from 2018-2020 → prices rose 30-50% from 2021-2022.

Concern 2: Can the Open-Source Flywheel Be Sustained?

Open-Source Leader Team R / Model B are currently supported by "public goodwill + national strategic support." If policy support weakens or the team faces greater commercialization pressure → open-source licensing could tighten. Analogy: Red Hat / MongoDB / Elastic all took the path of "open-source → commercial tightening."

Concern 3: Sustainability of the Compliance Premium

The current 30-100% compliance/private deployment premium is based on "domestic substitution + data security" policies. If overseas models are re-permitted entry into China → the compliance premium will be compressed.

6.2 Three Vulnerabilities of the Closed Loop

Vulnerability 1: Window of Opportunity for Single-Point Models — Model A currently has 95% penetration in short dramas → industry saturation; overseas video models following up with 3rd generation → how long can the moat last? Assessment: Video model opportunity window is 18-24 months; continuous investment in next-gen models is needed.

Vulnerability 2: Homogenization of Coding Models — Model C is currently the Coding leader, but overseas models + Cursor / Devin are all competing; once overseas models are re-permitted, the Coding leadership position could rotate. Assessment: Coding window is 12-18 months; must bind the developer ecosystem.

Vulnerability 3: Price War Quagmire for General LLMs — Domestic open-source 7B / 70B prices are already as low as 0.5-2 RMB/million tokens; limited room for further price cuts (already approaching marginal cost). Assessment: The general LLM track will consolidate to 3-5 oligopolies within 2 years, ending the price war.
