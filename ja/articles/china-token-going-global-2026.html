中国トークン海外進出 2026

1.2% から 61% へ——シェアの先にある真の問題：誰が海外展開し、どう収益化し、持続可能か

📅 2026-06-11 ⏱ ~10 min read 🏷️ Token Going Global · OpenRouter · MaaS · オープンソース大規模モデル
Token 海外進出 OpenRouter 61% オープンソース大規模モデル MaaS グローバル市場
EN Summary：Chinese open-source LLMs grew from 1.2% to 61% OpenRouter token share in 18 months. This report dissects who is actually going global, which markets matter, the pricing war at 1/30 of OpenAI, the 4 proven monetization paths, and 3 counter-intuitive findings (the real share is probably ~45% globally when AWS/Azure/Vertex are included; the biggest winners are middle-layer players like Together AI, not the model companies; open-source is the biggest lever AND the biggest risk).
⚠️ 核心結論：中国モデルが OpenRouter で 61% シェアを獲得しているのは表面的な現象である。実際のグローバルシェアはおそらく 45% に近い（AWS Bedrock / Azure / Vertex を含めた場合）。最大の勝者は最速で走るモデル企業ではなく、Together AI や OpenRouter のような「海外で中継役を担う中間層」である。オープンソース＋極限的低価格の双発エンジンはすでに天井に近づいており、次の段階は SaaS＋ハードウェアバンドルのフルスタック戦争である。

1概念の整理：「トークン海外進出」とは何か

よく混同される3つの概念：

概念意味代表例
モデル海外展開重みを Hugging Face に公開オープンソース先頭 R / トップ総合クラウド B / スタートアップ K / Z
API海外展開海外での推論サービス展開総合クラウド B 国際サイト / 動画クラウド A 国際サイト / Together AI
トークン海外展開サードパーティプラットフォーム経由の間接利用OpenRouter / Vercel AI Gateway / Cloudflare AI

📌 本当にシェアを決めるのは第3の概念

開発者は中国のクラウドアカウントを登録しない——彼らは OpenRouter 上でワンクリックでAPIを切り替える。海外展開は販売の問題ではなく、インフラの問題である：重み＋推論計算能力＋OpenAI互換インターフェース＋Stripe決済の4点セットを最初に整えた者が、受動的トラフィックを獲得する。

2プレイヤーマップ：5段階の階層化

「海外進出の実際の深度」に基づき5段階に分類（社名は一律コードネーム、対応関係は内部資料にて）：

🎯 第1段階 — グローバル開発者の恩恵を真に享受（海外コール率5%以上）

  • オープンソースリーダーチーム R：V3発表7日でOpenRouterトップに；MIT + OpenAI互換 + 極限的低価格の「三連打」；海外コール率60%以上
  • トップクラウド総合 B：モデルマトリクスが最も充実（0.6B–235Bフルサイズ + マルチモーダル）；HF累計ダウンロード4,200万回；OpenAI/Anthropic/Vertexの3つのSDKに対応
  • AIスタートアップ K：ロングコンテキスト（200K以上）+ ツール呼び出しがコア差別化要因；GitHub Stars 21K以上

📊 第2段階 — 認知されているがコール量は未爆発

  • AIスタートアップ Z：HFトレンドに長期ランクイン；海外ブランド認知度は低い；国内83%値上げで収益化に成功、海外の有料転換率は1/5
  • AIスタートアップ Y：ロングコンテキスト + ツール呼び出しが強み；商業化は依然模索中
  • AIスタートアップ S：マルチモーダルシーンで採用あり；規模は小さい

🌐 第3段階 — チャネル海外進出（モデルではなくコールを提供）

  • トップ動画クラウド A：動画生成モデル + 親会社の海外エコシステム波及；直接Runway / Pikaに対抗
  • 中国クラウド事業者 C：C社の国際サイトを通じて間接的に海外展開；東南アジアのゲーム/金融にサービス提供

⚙️ 第4段階 — インフラ海外進出

  • AIコンピューティングプラットフォーム T（Together AI）：中国バックグラウンド、北米最大の独立推論クラウド；Llama / Q / D / G系をホスティング
  • 大規模モデルルーティングゲートウェイ G（OpenRouter初期パートナー）：統一API + 自動fallback + 課金

5収益化パス：4 つの実証済みモデル

📦 パス A — API直販（大手クラウドベンダー）

収益 1 億ドル～50 億ドル/年；粗利率 30～50%；シンガポール/フランクフルト/バージニアの複数リージョンに展開；最大の課題は現地BDと中米関係下での顧客信頼

🔀 パス B — アグリゲーションプラットフォームの収益分配（スタートアップチーム）

収益 1,000 万ドル～5 億ドル/年；粗利率 50～70%；代表例は Together AI、Fireworks AI、OpenRouter；課題はモデルの反復速度

💼 パス C — 業界向けSaaS（最も収益性が高いが最も難しい）

収益 5 億ドル～100 億ドル/年；粗利率 60～80%；法務AI/医療AI/教育AIの3つの実例（ARR 3,000 万ドル / 単一プロジェクト 8,000 万ドル / 5 年で 2 億ドル）

🖥️ パス D — ハードウェア + モデル バンドル（IDC / チップメーカー）

収益 1 億ドル～30 億ドル/年；粗利率 20～35%；チップ + サーバー + 光モジュールメーカーが中国製モデルを組み合わせて東南アジア / 中東 / ラテンアメリカへ販売

6常識に反する3つの発見

💡 発見1 — 実際のグローバルシェアはすでに45%近くに達している可能性

OpenRouterの公開データ61%には「AWS Bedrock / Azure AI Foundry / Vertex Model Garden / Vercel/Cloudflare Gateway によってプッシュされている」部分が含まれていない。非明示的な呼び出しを加えると、世界のLLM推論トラフィックにおける中国製モデルの実際のシェアはすでに45%近くに達している可能性がある。

💡 発見2 — 最も成功しているのは中間層

Together AI は年率換算 5 億ドルのARR、粗利率 60%超であり、本質的には中国のオープンソースモデルを米国に再販売している。OpenRouter の初期チームの半数は華人系である。Cloudflare Workers AI は中国製モデルを「低遅延の代替品」として欧米のSaaS企業に販売している。結論：海外市場向け収益の大部分は「海外で中継を担う華人系チーム」が獲得している——越境ECにおける「華僑倉庫」モデルのAI版である。

💡 発見3 — オープンソースは最大のレバレッジであると同時に最大のリスク

三つの地雷：(1) クローズドソースのA系/B系が中国のオープンソースモデルの出力を組織的に蒸留している；(2) 2025年、米議会は「モデルの重み」をBIS規制リストに加える議論を開始；(3) 開発者はOpenAI互換プロトコル + Stripe + AWSへのデプロイに慣れており、中国製モデルとの結びつきは実際には非常に低く、一夜で切り戻せる。真の勝者は「オープンソース + クローズドソース + SaaS + ハードウェア」のフルスタック展開ができる者である。

7AI海外進出の最前線に関する判断

判断 1 — 顧客の購買行動は2層に二分される
• 個人開発者 / アーリーステージのスタートアップ：トークン課金 → 中国モデル = デフォルトオプション
• 中規模SaaS / エンタープライズ企業：ビジネス成果ベースの課金 → 中国モデル = 代替案 + 交渉材料
販売戦略は2系統に分ける必要がある：開発者向け（GitHub / Discord / HN）と、購買意思決定者向け（コンプライアンス + データ常駐 + 現地事例）
判断 2 — 3つの過小評価されている市場機会
1. 東南アジア多言語カスタマーサポート（インドネシア語 / ベトナム語 / タイ語）
2. ラテンアメリカのスペイン語プログラミング教育（メキシコ / ブラジル K12 + 職業訓練）
3. 中東アラビア語 + イスラム文化シーン（金融コンプライアンス / 教法相談 / コンテンツモデレーション）
判断 3 — リバース海外進出こそが本当に儲かる
国内MaaS事業者の国内収入は年間100〜150億元、海外API収入はその1/10未満。海外APIは価格競争 + 薄利多売の道。本当に儲かるのは、海外の開発者を国内の大規模エコシステム（WAIC / 雲栖 / 杭州 / 蘇州 / 深圳 AI起業園）に引き込むことである。
判断 4 — 警戒すべき法的闘争のシグナル
2025年第4四半期、ある米国主要AI企業が「中国のオープンソースモデルにトラフィックを奪われている」として法的措置を開始（訓練データのコンプライアンス追跡）。すべての海外大企業契約には「モデル切替条項」と「地政学的不可抗力条項」を盛り込む必要がある。

🎯 Key Takeaways

  • トークン海外展開 ≠ モデル海外展開 ≠ API海外展開：本当にシェアを決めるのは、サードパーティプラットフォーム（OpenRouter / Vercel / Cloudflare）上での間接的な呼び出しであり、中国クラウドベンダーの直接販売ではない
  • 61%は見かけ上の数字、実際の比率は45%に近い可能性：OpenRouterにはAWS Bedrock / Azure / Vertex上の中国モデルの呼び出しが含まれていない
  • 最大の勝者は中間層：Together AI（ARR 5億ドル、粗利益率60%）は「中国のオープンソースモデルを米国に売り戻す」ことが、直接海外展開するより100倍容易である
  • 1/30の価格競争はすでに常態化：キャッシュヒット時 0.014ドル/100万トークン、OpenAIバッチAPIより1桁低い
  • 4つの収益化経路：API直接販売 / アグリゲーション収益分配 / 業界SaaS / ハードウェアバンドル —— 「フルスタック展開」だけが最終形
  • 3つの過小評価された機会：東南アジア多言語カスタマーサポート / ラテンアメリカプログラミング教育 / 中東アラビア語シーン
  • オープンソースは最大のレバレッジであり最大のリスクでもある：蒸留 + BIS規制 + 低結合度、3つの地雷が待ち受ける
